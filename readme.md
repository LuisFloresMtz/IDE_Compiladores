# IDE Compiladores

IDE de escritorio para análisis léxico, sintáctico, semántico y generación de código intermedio, construido con Python y PySide6.

---

## Requisitos

- Python 3.11+
- pip

---

## Instalación

### 1. Clonar el repositorio

```bash
git clone https://github.com/LuisFloresMtz/IDE_Compiladores
cd ide-compiladores
```

### 2. Crear el entorno virtual

```bash
python -m venv venv
```

### 3. Activar el entorno virtual

**Windows:**
```bash
venv\Scripts\activate
```

**macOS / Linux:**
```bash
source venv/bin/activate
```

### 4. Instalar dependencias

```bash
pip install -r requirements.txt
```

---

## Ejecución

```bash
python main.py
```

## Desactivar el entorno virtual

```bash
deactivate
```
